# Pañca-Pramāṇa Boundary Checker

**Use this template** to sort every claim in your research project into one of the **five Nyāya means of valid knowledge** (pramāṇas). Each pramāṇa has its own evidence-rules. Cross-pramāṇa claim-promotion (e.g. asserting an analogy as a derivation) is the most common methodological error in cross-cultural / cross-disciplinary research.

This is the **firewall** that keeps the five evidence-types from contaminating each other.

---

## ◈ The 5 pramāṇas

| # | Pramāṇa | Sanskrit | Means | Evidence rule |
|---|---|---|---|---|
| 1 | **Pratyakṣa** | प्रत्यक्ष | direct perception / measurement | physically observable, repeatable, independently verifiable |
| 2 | **Anumāna** | अनुमान | formal inference | logical derivation from premises with explicit rules |
| 3 | **Upamāna** | उपमान | analogy / homological mapping | similarity-based reasoning — explicit source + target, explicit limits |
| 4 | **Śabda** | शब्द | authoritative testimony | cited lineage, traceable to source, source's authority bounded |
| 5 | **Arthāpatti** | अर्थापत्ति | postulation from circumstance | what must have been wrong for the surviving claims to be coherent |

## ◈ Common cross-pramāṇa errors (forbidden without explicit bridge-evidence)

These promotion-errors are the most common in cross-cultural research:

1. **Upamāna → Anumāna** (analogy asserted as derivation)
   *Example (forbidden):* "Sacred geometry resembles X formal structure, therefore the geometry IS X."
   *Correct:* "Sacred geometry has formal resemblance to X. The resemblance is upamāna-grade evidence. Formal derivation of X-properties requires anumāna and has not been done."

2. **Śabda → Pratyakṣa** (testimony asserted as observation)
   *Example (forbidden):* "The Vedas state Y, therefore Y is measurable."
   *Correct:* "The Vedas state Y. The textual evidence is śabda-grade. Whether Y is physically measurable is a separate pratyakṣa question and requires its own experiment."

3. **Anumāna → Pratyakṣa** (inference asserted as observation)
   *Example (forbidden):* "The math predicts Z, therefore Z is observed."
   *Correct:* "The math predicts Z. The prediction is anumāna-grade. Observation of Z is pratyakṣa-grade and requires an independent measurement."

4. **Pratyakṣa → Anumāna** (single observation asserted as general law)
   *Example (forbidden):* "I observed W once, therefore W always holds."
   *Correct:* "I observed W in this specific instance. Generalization to all instances requires anumāna with appropriate sample size."

5. **Arthāpatti → other** (postulation asserted as primary evidence)
   *Example (forbidden):* "Some premise must be wrong for the surviving claims to hold, therefore I know which premise is wrong."
   *Correct:* "Some premise must be wrong (arthāpatti). To identify WHICH premise requires fresh pramāṇa-1-through-4 evidence."

## ◈ Aggregation rule

For a project with N claims, compute the **pramāṇa-distribution table**:

| Pramāṇa | Count | % |
|---|---|---|
| Pratyakṣa | ... | ... |
| Anumāna | ... | ... |
| Upamāna | ... | ... |
| Śabda | ... | ... |
| Arthāpatti | ... | ... |
| Bridged (multi-pramāṇa with disclaimer) | ... | ... |
| **UNCLASSIFIED** | ... | ... |

**A healthy project has 0 unclassified claims and explicit bridge-disclaimers on any multi-pramāṇa claim.**

In the KAAL framework, the distribution is roughly:
- Pratyakṣa (hardware): ~5%
- Anumāna (algebra): ~30%
- Upamāna (cultural naming): ~50% (Tier-W)
- Śabda (classical citations): ~10%
- Arthāpatti (HALAHALA retractions): ~5%

## ◈ Operational rule

When you assert a claim:

1. **State its pramāṇa explicitly.** ("This is anumāna based on the substrate axioms.")
2. **State what is NOT claimed.** ("This is not pratyakṣa — no measurement has been made.")
3. **If you want to bridge pramāṇas, state the bridge-evidence explicitly.** ("To promote this upamāna to anumāna, I would need X. I have not done X.")
